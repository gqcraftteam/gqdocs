如何添加新模组文档

当您完成新模组的开发时，可以按照以下步骤添加文档：

步骤 1：创建模组文档文件

在 mods/ 文件夹中创建新的 HTML 文件，例如 mod-name.html

步骤 2：使用文档模板

参考现有模组的文档结构，包含以下部分：

  • 模组简介和概述
  • 功能特性列表
  • 安装说明
  • 使用教程
  • 常见问题

步骤 3：更新主文档页面

在此页面（docs.html）中添加新模组的卡片链接，使其出现在模组列表中。

<div class="mod-card">
  <div class="mod-header">
    <h3>🎮 您的模组名称</h3>
    <span class="mod-version">v1.0.0</span>
  </div>
  <p class="mod-description">模组描述</p>
  <div class="mod-features">
    <span class="feature-tag">功能标签</span>
  </div>
  <a href="mods/your-mod.html" class="mod-link">查看完整文档 →</a>
</div>

步骤 4：每个模组有独立页面

每个模组的完整文档都在独立的 HTML 文件中，这样便于管理和更新。
