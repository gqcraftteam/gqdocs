模组概述

示例模组 2 是一个专注于工具和自动化的 Minecraft 模组...

开发者： GQCraft Team

版本： 0.9.0

Minecraft 版本： 1.19.x - 1.20.x

Forge 版本： 43.0+

功能特性

  • 自动化工具： 简化重复任务
  • 高级工作台： 新的合成方式
  • 能量系统： 新的电力机制
  • 存储系统： 增强的物品存储

安装说明

前置要求

  • Minecraft 1.19 或更高版本
  • Minecraft Forge 43.0 或更高版本
  • 示例模组 1（可选，某些功能需要）

安装步骤

  1. 从官网下载最新版本
  2. 将 mod2-0.9.0.jar 放入 mods 文件夹
  3. 如需启用高级功能，请同时安装示例模组 1
  4. 启动游戏

使用教程

初次使用

安装模组后，您将获得一个新手指南物品。点击它可以查看快速入门教程。

工具使用

模组提供了多种工具来帮助您提高效率：

  • 扳手工具 - 用于调整和配置
  • 扫描仪 - 用于发现能量网络
  • 日志工具 - 用于调试

能量系统

模组使用通用能量系统 (RF/FE)，兼容其他使用同样标准的模组。

API 参考

对于模组开发者，示例模组 2 提供了强大的 API 支持。

主要接口

com.gqcraft.mod2.api.IEnergyHandler
com.gqcraft.mod2.api.IStorage
com.gqcraft.mod2.api.IAutomation
com.gqcraft.mod2.api.IRecipe

使用示例

IEnergyHandler handler = tile.getCapability(...);
if (handler != null) {
    int received = handler.receiveEnergy(1000, false);
}

常见问题

Q: 能量如何运输？

A: 使用专用的能量传导线连接设备，能量会自动流动。

Q: 如何扩展模组功能？

A: 可以通过 JSON 配置文件或使用提供的 API 来添加新的合成配方。

Q: 模组性能如何？

A: 该模组经过高度优化，与基础版本相比占用资源很少。

Q: 支持模组包吗？

A: 是的，完全支持包含在模组包中使用。
