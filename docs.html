快速开始

前置条件

  • 基本的 HTML/CSS/JavaScript 知识
  • 文本编辑器或 IDE
  • 现代化的网络浏览器

安装步骤

  1. 克隆或下载项目文件
  2. 在浏览器中打开 index.html
  3. 开始使用

API 参考

GET /api/docs

获取所有文档列表

响应示例：

{
  "status": "success",
  "data": [
    {
      "id": 1,
      "title": "介绍",
      "content": "..."
    }
  ]
}

POST /api/docs

创建新文档

请求示例：

{
  "title": "新文档",
  "content": "文档内容"
}

使用示例

示例 1：基础布局

创建一个简单的两栏布局：

<div class="container">
  <aside class="sidebar"></aside>
  <main class="content"></main>
</div>

示例 2：响应式网格

使用 CSS Grid 创建响应式布局：

.grid {
  display: grid;
  grid-template-columns: repeat(auto-fit, minmax(300px, 1fr));
  gap: 1rem;
}
